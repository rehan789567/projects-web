# 🚀 PROJECTS 2026 — Interactive Web Portfolio

> A visually rich, multi-experience portfolio built with **pure HTML, Tailwind CSS, and JavaScript** — no frameworks, no build tools, just clean frontend craftsmanship.

---

## 🌌 Overview

This repository contains **four fully interactive web applications**, each designed with a **distinct UI philosophy** and inspired by traditional programming logic translated into modern web experiences.

Crafted by **Muhammad Rehan**, this project focuses on:

* Creative UI/UX systems
* Smooth animations & micro-interactions
* Clean, responsive layouts
* Real-world logic implementations

---

## 🎯 Featured Projects

### 💳 Rehan Bank ATM *(Skeuomorphic UI)*

* Realistic ATM simulation
* Metallic textures & digital display feel
* Secure PIN interaction logic
* Classic interface reimagined for the web

---

### 🧮 Neo-Calc Precision *(Glassmorphism UI)*

* Frosted glass design with neon accents
* Smooth button interactions
* Calculation history (tape system)
* Minimal yet futuristic

---

### 🎮 Number Quest *(Claymorphism UI)*

* Fun number guessing game
* Soft UI elements & playful animations
* Dynamic feedback system
* Designed for engagement

---

### 🔐 AES-256 Vault *(Cyber / Hard-Tech UI)*

* Security terminal-inspired interface
* Biometric-style animations
* Matrix-like visual effects
* Simulated authentication system

---

## 🎨 Design Philosophy

Each project explores a different modern UI trend:

| Project    | Design Style    |
| ---------- | --------------- |
| ATM        | Skeuomorphism   |
| Calculator | Glassmorphism   |
| Guess Game | Claymorphism    |
| Vault      | Cyber / Tech UI |

---

## ⚙️ Tech Stack

* **HTML5**
* **Tailwind CSS (via CDN)**
* **Vanilla JavaScript**
* **Lucide Icons**
* **Google Fonts (Plus Jakarta Sans)**

> ⚡ No frameworks. No bundlers. No dependencies.
